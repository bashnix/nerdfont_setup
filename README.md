Nerdfont installer
==================

I got tired of installing NerdFonts manually every time I distrohop, so I made this little script.

Usage
------------------

1. Clone this repository

```
git clone https://github.com/bashnix/nerdfont_setup.git
cd nerdfont_setup
```

2. List available fonts with

```
sudo ./setup list
```

3. Install fonts with

```
sudo ./setup install ExampleFont1 ExampleFont2
```
If you want to install everything, use
```
--all
```

4. If you want to uninstall later, use

```
sudo ./setup uninstall ExampleFont1 ExampleFont2
```
Also works with all
